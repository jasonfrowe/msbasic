Microsoft BASIC for the Picocomputer 6502.
Core interpreter by Bill Gates and Paul Allen.
Floating-point math routines by Monte Davidoff.
Released in 2025 with an MIT license:
https://github.com/microsoft/BASIC-M6502

Key changes and notes:
  CTRL-C stops a running program or listing.
  Type a line number then press TAB to edit a line.
  LOAD and SAVE file format is plain ASCII.
  Keywords and variable names may be entered in lower case.
  RND() is seeded with hardware entropy at cold-start.
  CAPS <expr> selects a caps mode; see launch options.

Launch arguments:
  -c0     caps mode off (case as typed)
  -c1     caps mode on  (default - letters always upper case)
  -c2     caps mode invert (swap case)
  <name>  program is loaded and run automatically

Graphics commands (early preview):
  Canonical style uses full words (MODE, CLEAR, PSET, etc.).
  GFX MODE,0             Return to console canvas
  GFX MODE,180           320x180 bitmap mode (8-bit color index)
  GFX MODE,180T          320x180 bitmap + 7 console text rows at bottom
  GFX MODE,240           320x240 bitmap mode (4-bit color index)
  GFX MODE,m,bp,cp       Mode with bitmap plane bp and console plane cp
  GFX PSET,X,Y,C         Plot pixel at X,Y with color C
  GFX LINE,X1,Y1,X2,Y2,C Draw a line from X1,Y1 to X2,Y2
  GFX BOX,X1,Y1,X2,Y2,C  Draw an outline rectangle
  GFX CIRCLE,X,Y,R,C     Draw a circle centered at X,Y with radius R
  GFX HLINE,X1,Y,X2,C    Draw a horizontal line from X1 to X2
  GFX VLINE,X,Y1,Y2,C    Draw a vertical line from Y1 to Y2
  GFX RECT,X1,Y1,X2,Y2,C Draw a filled rectangle
  GFX CLEAR              Clear bitmap only (graphics mode required)
  GFX CCLEAR             Clear console terminal only
  GFX CLEAR,ALL          Clear bitmap and console terminal
  GFX RESET              Restore default graphics state and console
  GFX STATUS             Show current mode and layer configuration
  GFX BITMAP,P           Set bitmap plane to P (0..2)
  GFX OVERLAY,P          Set console overlay plane to P (0..2)
  GFX OVERLAY,255        Disable console overlay
  Single-letter aliases (M/C/P/H/L/V/R/B/O/S) remain for compatibility only.

Audio commands (early preview):
  All audio commands use the SFX keyword with a subcommand.
  SFX n              Play one-shot effect n (auto-voice)
  SFX n,v            Play one-shot effect n on voice v (0..2)
  SFX STOP           Silence all voices immediately
  SFX VOL,n          Set master SFX volume 0..255
  SFX BANK,n         Select SFX sound bank
  SFX STATUS         Show current audio state
  SFX TEMPO,bpm      Set music tempo 30..300 BPM
  SFX VOICE,v,inst   Set voice v (0..2) to instrument id
  SFX NOTE,v,p,d     Play note on voice v; pitch 0..127, dur 1..255 ticks
  SFX NOTE,v,p,d,vel As above with velocity 0..15

File open/close and directory services will be added in the future.
For now, press CTRL-ALT-DEL or send BREAK to drop into the RP6502
monitor where you can list and change directories or unlink files.
Type RESET to warm-start BASIC, preserving program memory.
